# C++ Learning Projects

This repository contains small C++ projects I built while learning programming, mathematics, and physics.  
My goal is to improve my problem-solving skills by turning math and physics concepts into code.

## Projects

- **Circumference Calculator**  
  Calculates the circumference of a circle using user input.

- **RPG Character Creator**  
  A simple character creation program using variables, input, and formatted output.

- **Mini Choice Game**  
  A basic text-based choice game using conditional statements.

- **Mini Choice Game v2**  
  An improved version of the choice game with more logic and structure.

- **Projectile Analyzer**  
  Uses quadratic equations to analyze projectile motion, including height over time.

- **Quadratic Formula Calculator**  
  Solves quadratic equations using the quadratic formula.

- **Vector Component Calculator**  
  Calculates the x- and y-components of a vector using magnitude, angle, trigonometry, and radians conversion.
